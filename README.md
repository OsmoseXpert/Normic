# Normic Starter (Vite + Kevin Powell / CUBE CSS basis)

## Installeren
1. Open deze map in VS Code
2. Open een terminal in deze map
3. Run:

```bash
npm install
npm run dev
```

## Wat zit erin?
- `index.html` (header + hero + features + testimonials + footer)
- `style.css` (Andy Bell reset + CSS variables + utilities/composition/blocks)
- `main.js` (mobile menu toggling via `data-visible`)

## Notities
- Het mobiele menu togglet via `data-visible="true/false"` op de `<ul>`.
- Buttons werken via data-attributen: `data-type="accent|neutral"` en optioneel `data-shadow="true"`.
